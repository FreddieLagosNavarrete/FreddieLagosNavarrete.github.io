# Portafolio Freddie Lagos - título profesional agregado

Cambios:
- Título Profesional de Ingeniero de Ejecución en Informática agregado como
  primer documento de la biblioteca.
- Filtro independiente: "Título profesional".
- Formación UCINF actualizada con "Aprobado con Distinción".
- Copia pública del PDF con el RUT protegido.
- Vista previa WEBP optimizada.
- Recursos versionados para evitar caché de GitHub Pages.

## Publicación
1. Elimina los archivos anteriores del repositorio.
2. Sube todo el contenido del ZIP a la raíz.
3. Verifica que `index.html` quede en la raíz.
4. Espera el despliegue.
5. Actualiza con Ctrl + F5.
